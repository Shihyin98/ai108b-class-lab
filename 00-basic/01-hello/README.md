# 1. area.py
## 1.1- Terminal
```
D:\ai108b\00-basic_w1-0305-Python\01-hello> py .\area.py
```

## 1.2- Output
```
Area is 10          # Output 1:  area
Perimeter is 14     # Output 2:  perimeter
```

## 1.3- Solution
* Initial-Variables:
  * length = 5
  * breadth = 2
<br>

* Output 1:
  * ``Area is 10``
    area = length * breadth
<br>

* Output 2:
  * ``Perimeter is 14``
    perimeter = 2 * (length + breadth)
<br>

## 1.4- Explain
* 不需要加「;」在每行程式碼後。
* Python 不需要事先宣告變數的類型就可以直接賦值。[[1]](-)
  Python會自動判斷你賦予的值的類型，並給變數也標籤上同樣的數據類型。
